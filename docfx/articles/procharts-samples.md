# ProCharts Samples

The sample app includes several pages that showcase ProCharts features.

Run the sample app:

```bash
dotnet run --project src/DataGridSample/DataGridSample.csproj
```

## Sample pages

- Charting
  - Basic chart types, styles, and interactions.
- Pivot Chart
  - Pivot-backed chart data.
- Pivot Calculated Measures
  - Formula-driven measures feeding the chart.

## Tips

- Use the chart sample controls to toggle series types and styling.
- Combine formulas and pivot aggregations to validate Excel parity behavior.
